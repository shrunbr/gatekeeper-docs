# Question Commands

In this section you will find a command reference for the `/question` commands. If you require additional assistance, reach out to us on our support discord.

## add

**Command:** `/question add`

**Options:** `type`, `text`, `placeholder`, `min_length`, `max_length`, `required`

**Description:** Add a new question.

## delete

**Command:** `/question delete`

**Options:** `question_id`

**Description:** Deletes a question.

## edit

**Command:** `/question edit`

**Options:** `question_id`, `type`, `text`, `placeholder`, `min_length`, `max_length`, `required`

**Description:** Edits a question.

## list

**Command:** `/question list`

**Options:** `None`

**Description:** Lists all configured questions for a guild.

## reorder

**Command:** `/question reorder`

**Options:** `question_id`, `new_order`

**Description:** Reorder the questions.
