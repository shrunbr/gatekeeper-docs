# Setup Commands

In this section you will find a command reference for the `/setup` commands. If you require additional assistance, reach out to us on our support discord.

## approver-role

**Command:** `/setup approver-role`

**Options:** `role`

**Description:** Setup the role allowed to approve applications.

## manager-role

**Command:** `/setup manager-role`

**Options:** `role`

**Description:** Setup the role allowed to manage Gatekeepers configuration. This role is also allowed to approve applications.

## member-role

**Command:** `/setup member-role`

**Options:** `role`

**Description:** Setup the role granted to members upon application approval.

## view

**Command:** `/setup view`

**Options:** `None`

**Description:** View the current configuration of Gatekeeper.

## help

**Command:** `/setup help`

**Options:** `None`

**Description:** Get help on Gatekeeper.
